This README outlines what each function does in regards to binary trees

0-binary_tree_node   	       - creates a new binary tree nodei

1-binary_tree_insert_left      - inserts a node as the left child of another node. if one already exists, it will replace that node and the old node will become the left-child of the new node

2-binary_tree_insert_right     - inserts a node as the right child of another node. If one already exists, it will replace thta node and the old node will become the right-child of the new node

3-binary_tree_delete 	       - deletes a whole tree

4-binary_tree_is_leaf 	       - checks if a node is a leaf

5-binary_tree_is_root 	       - checks if a node is root
